### after-class problem
 现有一笼子，里面有鸡和兔子若干只，数一数，共有  
头14个，腿38条，求鸡和兔子各有多少只？
#### 小学解法
假如全是鸡，14只，腿有28条，38-28=10。每只兔子腿比鸡多2条，10/2=5，故应该有5只兔，9只鸡。
#### 中学解法
列方程，设鸡有x只，兔有y只：
x+y=14,2x+4y=38 >>> x=9,y=5
